# ai-rag-agentic-systems-guide
Projeto focado em AI Engineering, explorando RAG (Retrieval-Augmented Generation) e Agentic Systems utilizando NotebookLM. O objetivo é demonstrar domínio em arquiteturas modernas de IA, engenharia de prompts e construção de sistemas inteligentes alinhados às práticas de mercado utilizadas por Big Techs.
